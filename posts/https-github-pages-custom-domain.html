HTTPS on Github Pages custom domains

Inability to use HTTPS on a custom domain hosted by Github Pages has been an issue for years with no resolution in sight. Today, I stumbled upon a blog post about fly.io and it blew my mind. I thought, "What a sparse tutorial! It can't be that easy."

Well, it is.

Setting up HTTPS for my custom domain for my Github Page took 5 minutes and $0. See my comment on the Github Issue for how it works. Happy encrypting!
